# Windows 端 AI 病历书写图文教程

> 规培-medlift — 把病历重担从规培医生肩上抬走

---

## 第一步：搭建局域网环境（手机热点 + WiFi 接收器）

### 所需设备

- **手机（开启热点）**：iPhone / 安卓均可，推荐使用备用机或主号热点的副卡
- **WiFi 接收器**（约十几元，淘宝/拼多多搜索"USB WiFi 接收器"）
  - 插入科室电脑的 USB 接口
  - 搜索并连接手机发出的 WiFi 热点

### 连接步骤

```
手机开启热点
    ↓
WiFi 接收器插入科室电脑 USB
    ↓
电脑搜索并连接手机热点
    ↓
电脑与手机处于同一局域网下
    ↓
可使用 LocalSend / Feem 传输文件
```

> 📡 有了局域网环境，手机和电脑可以互通，方便传输 prompt 内容、接收生成的病历文本，也可以在电脑上直接访问 AI 网页服务。

---

## 第二步：使用 Prompt 生成器构建输入

打开本仓库中的 `ai病历通用结构化prompt.html` 文件（或双击下载的 HTML 文件在浏览器中运行）。

根据患者信息填写以下内容：

<img src="https://raw.githubusercontent.com/liuguangyihuanchang/guipei-medlift/main/docs/图片005.jpg" width="800" />

| 填写项 | 说明 |
|--------|------|
| **患者姓名 / 性别 / 年龄** | 基本信息（选填，但有助于 prompt 优化） |
| **入院 / 手术 / 出院日期** | 关键时间节点 |
| **既往病史** | 过敏史、手术史、慢性病等 |
| **在院检验结果** | 可直接粘贴化验单文字 |
| **在院检查报告** | CT、MRI、X光、B超描述 |
| **当前医嘱 / 用药方案** | 当前住院医嘱 |

填写完成后，点击**「生成 Prompt」**，将生成的完整 Prompt 文本**全选复制**。

---

## 第三步：调用 AI 生成病历

### 选择任一 AI 服务

手机开热点后，电脑连接热点，即可访问以下任一 AI 平台网页端（无需 VPN）：

| AI 服务 | 网址 |
|---------|------|
| **智谱清言**（GLM） | https://chatglm.cn |
| **DeepSeek** | https://chat.deepseek.com |
| **通义千问**（Qwen） | https://qwen.ai |
| **豆包** | https://doubao.ai |
| **MiniMax** | https://hailuoai.video |

### 操作流程

```
第1步   电脑连接手机热点，打开上述任一 AI 服务网页
         ↓
第2步   将第二步复制的 Prompt 完整粘贴到 AI 对话框
         ↓
第3步   提交，等待 AI 生成完整病程记录
         ↓
第4步   将生成的病历内容复制粘贴回医院病历系统
```

---

## ⚠️ 患者隐私保护提醒

- 生成的病历内容在正式提交前须经主治医师审核确认
- 请勿将含真实患者信息的截图或内容随意传播
- AI 生成的病历仅供参考，最终临床决策由医师负责

---

## 💡 小贴士

- **WiFi 接收器**建议选支持 2.4G/5G 双频的，稳定性更好
- 如果医院科室 WiFi 管控不严，也可以直接连接科室 WiFi 跳过热点步骤
- AI 生成的内容建议仔细核对医嘱日期、用药剂量等关键信息
- Prompt 中的检验检查结果可以直接粘贴 OCR 识别后的文字，省去手工录入
