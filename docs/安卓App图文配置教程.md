## 第一步：前置大模型账户的创建及 API Key 的申请

### 1.1 创建阿里云账号

1. 访问 [阿里云大模型服务平台百炼](https://bailian.console.aliyun.com/)，使用支付宝或淘宝账号登录
2. 建议完成**个人实名认证 + 人脸验证**（部分模型需要）

### 1.2 进入控制台

点击右上角进入**大模型服务平台百炼控制台**：

![图001：进入百炼控制台](docs/图片001.jpg)

### 1.3 创建 API Key

点击左下角 **API-Key** 管理入口：

![图002：API-Key 管理入口](docs/图片002.jpg)

点击**创建 API-Key**，复制并妥善保存，后续需要填入 App 中：

![图003：创建并复制 API Key](docs/图片003.jpg)

## 第二步：下载并安装 App

1. 访问 GitHub 仓库：https://github.com/liuguangyihuanchang/guipei-medlift
2. 跳转到 **Releases** 页面，下载 App 安装包（Android APK）及 Windows 版所需文件
3. 将 APK 安装到手机上

## 第三步：在 App 内配置 API

1. 点击主界面 **「医疗病历记录」** 进入病历书写功能页面

2. 按照以下内容填入配置（API Key 来源自第一步）：

| 配置项 | 填写内容 |
|--------|---------|
| **API Key** | 第一步创建的阿里云百炼 API Key |
| **URL** | `https://dashscope.aliyuncs.com/compatible-mode/v1`（阿里云默认） |
| **模型名称** | 参考下方截图中的模型配置 |
